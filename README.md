# mirai-class-notice

> 🥳 这是一款基于 mirai 的班级 QQ 群课程通知机器人，拥有简单的 web 管理页面，尽量实现一键安装简易操作等功能。

> **版本：** v0.1 Beta

### 使用 🍗
[mirai-class-notice 上课通知机器人简单搭建部署使用教程](/docs/use.md)

### 功能 🦁

- 用户登陆
- 上课时间设定
- 课程表添加
- 系统状态
- 系统设置
- 定时任务
- 修改管理员密码
- 暂停课程通知
- 修改课程信息
- 删除课程
